# ChildCare - Missing Child Reporting System

## Overview

A web-based application developed using Java, JSP, Servlet, JDBC, MySQL and Apache Tomcat for reporting and tracking missing child cases.

## Modules

### Citizen Module
- Registration
- Login
- Report Missing Child
- Upload Child Image
- Track Report Status

### Police Module
- View Reports
- Update Status
- Verify Cases

### Admin Module
- Manage Citizens
- Manage Police
- Manage Reports

## Technologies Used

- Java
- JSP
- Servlet
- JDBC
- MySQL
- Bootstrap
- Apache Tomcat
